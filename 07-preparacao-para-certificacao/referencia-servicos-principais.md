# ⚡ Referência Rápida: Serviços AWS Cloud Practitioner

Esta tabela é um guia de consulta rápida dos serviços mais cobrados na certificação CLF-C01, organizados por Categoria.

| Categoria | Serviço | Descrição e Função Principal |
| :--- | :--- | :--- |
| **COMPUTE** | **Amazon EC2** | Capacidade de computação virtual (IaaS - Máquinas Virtuais). |
| **COMPUTE** | **AWS Lambda** | Computação Serverless (sem servidor) baseada em eventos. |
| **COMPUTE** | **AWS Auto Scaling** | Adiciona ou remove automaticamente instâncias EC2 com base na demanda. |
| **COMPUTE** | **ELB** (Load Balancer) | Distribui o tráfego entre instâncias para alta disponibilidade. |
| **STORAGE** | **Amazon S3** | Armazenamento de Objetos (Object Storage) escalável e de alta durabilidade (sites estáticos). |
| **STORAGE** | **Amazon EBS** | Armazenamento de Bloco (Block Storage) persistente, usado como disco para instâncias EC2. |
| **STORAGE** | **Amazon Glacier** | Armazenamento de baixo custo para arquivamento e backup de longo prazo. |
| **DATABASE** | **Amazon RDS** | Banco de dados Relacional gerenciado (MySQL, PostgreSQL, etc.). |
| **DATABASE** | **Amazon DynamoDB** | Banco de dados NoSQL totalmente gerenciado, escalável e de baixa latência. |
| **DATABASE** | **Amazon Redshift** | Data Warehouse para análise de grandes volumes de dados. |
| **NETWORK** | **Amazon VPC** | Rede Virtual isolada na AWS (seu data center virtual). |
| **NETWORK** | **Amazon Route 53** | Serviço DNS (Domain Name System) web escalável e de alta disponibilidade. |
| **NETWORK** | **Amazon CloudFront** | Rede de Distribuição de Conteúdo (CDN) que usa Edge Locations para reduzir latência. |
| **SECURITY** | **AWS IAM** | Gerenciamento de Identidade e Acesso (Usuários, Grupos, Roles e Políticas). |
| **SECURITY** | **Security Groups** | Firewall no nível da **Instância** (com estado). |
| **SECURITY** | **NACL** | Firewall no nível da **Sub-rede** (sem estado). |
| **SECURITY** | **AWS CloudTrail** | Log de todas as ações (chamadas de API) feitas na conta, para auditoria. |
| **SECURITY** | **Amazon GuardDuty** | Serviço de detecção de ameaças e atividades maliciosas. |
| **SECURITY** | **AWS WAF** | Web Application Firewall (WAF) para proteger contra injeção de SQL e XSS. |
| **GOVERNANÇA** | **AWS Trusted Advisor** | Fornece recomendações de segurança, otimização de custos e performance. |
| **GOVERNANÇA** | **AWS Config** | Monitora e avalia a conformidade da configuração dos recursos ao longo do tempo. |
| **GOVERNANÇA** | **CloudFormation** | Infraestrutura como Código (IaC) - Criação e provisionamento de recursos automatizados. |
